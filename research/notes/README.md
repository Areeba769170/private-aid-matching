# Research notes

Short, honest summaries of papers and sources as they're read — one file per
source. The point is to build on existing work (biometric template protection,
fuzzy / threshold PSI, secure two-party computation) rather than reinventing it.

Suggested format per note: full citation, the problem the work solves, its core
idea in plain language, its limitation / what it leaves open, and how it relates
to this project.
